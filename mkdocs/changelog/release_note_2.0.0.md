# Version 2.0.0

## **General**
- **IMPORTANT**: Zen BBQ add-on has changed from [Legacy add-on to Extension](https://docs.blender.org/manual/en/latest/advanced/extensions/addons.html#legacy-vs-extension-add-ons).
- **Node System Overhaul:** replaced the node architecture with an upgraded, user-proposed node structure for cleaner material graphs and improved flexibility.
- **Texture Baking Module:** complete, streamlined environment for configuring and baking texture maps (such as Normal, AO, and Curvature) directly inside Blender. It integrates object management, rendering engine settings, quality presets, and file output path templates into a single vertical layout.
- **Combined N-Panel:** reorganized the N-Panel UI into a combined panel, consolidating all floating addon panels into structured layout tabs.
- **Shader and Image Preview Engine:** powerful toolset within Zen BBQ designed to manage how your objects and bevels are visualized in the Blender viewport.
- **Normal Preview Studio:** classic three-point lighting rig paired with a camera and a target Empty to help inspect shading, normal orientation, and procedural bevels under optimal lighting conditions.
- **Normal Map Tools:** a suite of utilities designed to fine-tune surface normals and prevent projection artifacts.
- **Explode System:** prevent unwanted bleeding or projection artifacts.
- **Blender 5.2 Support:** Full compatibility updates and adjustments for Blender 5.2.
